# Lock n More Monorepo

This repository contains the full Lock n More system:

- backend/Toto-Backend: Express + TypeScript API
- frontend/Lock-n-More-Admin-FE-: React + Vite admin frontend
- docs: project documentation

## Local Development

Backend:
1. cd backend/Toto-Backend
2. npm install
3. npm run migrate
4. npm run dev

Frontend:
1. cd frontend/Lock-n-More-Admin-FE-
2. npm install
3. npm run dev

## Build Validation

Backend:
- cd backend/Toto-Backend
- npm run build

Frontend:
- cd frontend/Lock-n-More-Admin-FE-
- npm run build
